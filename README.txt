


















Для начала работы нужно повторить в последовательности следующее:
 Установить на !чистую! (если из нижеописаного ничего не поставлено) робочую машину: 
 1)nodeJS -отсюда : https://nodejs.org/en ;
 2)консоль git- отсюда : https://git-scm.com/ ;
 3)редактор кода Visual Studio Code можно отсюда: https://code.visualstudio.com/ ;
        Для коректной работы следует запускать Visual в режиме админимстратора-
		  лучше всего раз указать это в свойствах ярлыка.
			установить в Visual Studio Code плагины:
			1)Live server;
			2)Path Autocomplete; -после установки в  любом открытом редакторе, набрать F1 (FN+F1)-вызвать поиск-
					в поиске прописать "Open Settings (JSON) и в самом файле в конце или где небудь прописать 

					"path-autocomplete.pathMappings": {
    				"@img": "${folder}/app/img",
    				"@scss": "${folder}/app/scss",
    				"@js": "${folder/app/js},"
 					 },
					
 4) Вызываем терминал (просто протянуть с низу ввверх строку состояния) переходим на терминал.
 В настройках (через + ) ставим терминал по умолчанию GIT BASH.

 5)Открыть в visual Studio Code папку с будующим проектом(обозвать можно как угодно -главное на латинице)
 6)В открытой папке должны лежать файлы с архива:
	папка: app
	файл:  package-lock.json
	файл:  package.json
	файл:  README.txt 

	7) Если на машине впервые запускается gulp -последний надо установить глобально в систему.
	Для этого в терминале прописываем команду: 'npm i gulp-cli -g'  и жмем enter
	8) для установки всех необходимых плагинов в сборке в терминале прописываем команду: 
	'npm i'  и жмем enter.
	9) ЗАПУСК И РАБОТА 
	 первым делом рекомендую положить в !ЛЮБОМ ФОРМАТЕ! ШРИФТЫ  необходимые для верстки в папку "app/fonts/"
	 и запустить  в терминале команду: gulp font ; нужно сделать один раз -gulp переформатирует все в нужный формат и 
	 положит все в папку dist/fonts/ также создаст в папке dist/glavnaya/scss/ файл fonts.scss в котором будут записаны 
	 все ШРИФТЫ.
	 10) Основной ЗАПУСК -команда 'gulp' 
	 11) дальше можно редактировать нужные файлы в папке app -сборщик будет автоматом сохранять все изменения в папке dist/
	 12) остановить работу 'ctrl+cancel '
	 13) посмотреть набор доступных команд - 'gulp --tasks'
